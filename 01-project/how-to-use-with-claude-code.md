# Dùng với Claude Code

## Thiết lập
1. Sao chép thư mục này vào dự án.
2. Đặt `CLAUDE.md` ở thư mục gốc.
3. Mở Claude Code tại thư mục gốc.
4. Yêu cầu đọc `CLAUDE.md`.

## Khi giao việc
Chỉ dẫn Claude đọc:
- Tài liệu nền tảng.
- Tài liệu thiết bị.
- Tài liệu thành phần.
- Mẫu nghiệp vụ liên quan.

Không yêu cầu đọc toàn bộ tài liệu cho mọi nhiệm vụ.
