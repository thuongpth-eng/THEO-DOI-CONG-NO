# Color System // Hệ thống màu

## Màu thương hiệu
| Token // biến | Giá trị | Mục đích |
|---|---:|---|
| `brand-primary` | `#60BB46` | Nút chính, hoàn thành, đã duyệt |
| `brand-accent` | `#0969A7` | Liên kết, thông tin, điểm nhấn |
| `nav-base` | `#4B4F55` | Thanh điều hướng |
| `danger` | `#E53935` | Lỗi, thất bại, xóa |
| `warning` | `#FFA726` | Chờ duyệt, cảnh báo |
| `muted` | `#9E9E9E` | Hủy, không hoạt động |

## Quy tắc
- Không dùng mã màu trực tiếp trong thành phần.
- Mọi màu phải đi qua Design Token // biến thiết kế.
- Trạng thái không được chỉ thể hiện bằng màu.
