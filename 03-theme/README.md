# Giai đoạn 03: Chủ đề và giao diện tối

## Mục tiêu
Chuẩn hóa phần chủ đề và giao diện tối để áp dụng thống nhất cho toàn bộ ứng dụng HPCons.

## Nguyên tắc
- Tuân thủ `CLAUDE.md`.
- Không tạo quy chuẩn mâu thuẫn với phần nền tảng.
- Mọi thay đổi phải kiểm tra trên PC // máy tính, Tablet // máy tính bảng và Mobile // điện thoại.
- Ưu tiên tái sử dụng.
