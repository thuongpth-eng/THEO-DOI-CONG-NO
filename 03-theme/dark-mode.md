# Dark Mode // Giao diện tối

## Màu nền
| Token | Giá trị |
|---|---:|
| `background` | `#0F1720` |
| `surface` | `#121C26` |
| `card` | `#18232E` |
| `elevated` | `#1E2B36` |
| `sidebar` | `#4B4F55` |
| `border` | `rgba(255,255,255,0.08)` |
| `divider` | `rgba(255,255,255,0.06)` |

## Màu chữ
- Chính: `#F5F7FA`
- Phụ: `#B8C0C8`
- Mô tả: `#8F9AA5`
- Vô hiệu hóa: `#66717C`

## Quy tắc
- Dark Mode là mặc định.
- Không dùng đen tuyệt đối trên toàn bộ màn hình.
- Không dùng trắng tuyệt đối trên diện tích lớn.
