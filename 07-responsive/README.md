# Giai đoạn 07: Tương thích đa thiết bị

## Mục tiêu
Chuẩn hóa phần tương thích đa thiết bị để áp dụng thống nhất cho toàn bộ ứng dụng HPCons.

## Nguyên tắc
- Tuân thủ `CLAUDE.md`.
- Không tạo quy chuẩn mâu thuẫn với phần nền tảng.
- Mọi thay đổi phải kiểm tra trên PC // máy tính, Tablet // máy tính bảng và Mobile // điện thoại.
- Ưu tiên tái sử dụng.

## Trạng thái
Đã khởi tạo trong HPCons Design System V1.0.
