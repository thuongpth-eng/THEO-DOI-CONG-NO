# Responsive Rules // Quy tắc tương thích đa thiết bị

## Breakpoint // điểm chuyển giao diện
- Mobile: `<768px`
- Tablet: `768px–1279px`
- Desktop: `>=1280px`

## Quy tắc chuyển đổi
- 4 cột → 2 cột → 1 cột.
- Sidebar cố định → thu gọn hoặc Drawer → Drawer.
- Table đầy đủ → Table rút gọn → Card List.
- Dialog giữa màn hình → Dialog rộng 80% → Full Screen hoặc Bottom Sheet.
