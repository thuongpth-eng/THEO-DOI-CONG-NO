# Button // Nút bấm

## Kích thước
- PC: 40–44px
- Tablet: 44–48px
- Mobile: 48px

## Biến thể
- Primary // chính: `#60BB46`
- Secondary // phụ: `#0969A7`
- Outline // viền
- Ghost // nền trong suốt
- Danger // nguy hiểm: `#E53935`

## Trạng thái
Default, Hover, Active, Focus, Disabled, Loading.

## Quy tắc
- Một khu vực chỉ nên có một nút chính.
- Hành động xóa phải xác nhận.
