# Coding Standard // Quy chuẩn lập trình

## Bắt buộc
- TypeScript strict // kiểm tra kiểu dữ liệu nghiêm ngặt.
- Không dùng `any` nếu không có giải thích.
- Không dùng inline style.
- Không hard-code // viết cứng màu, khoảng cách, bo góc.
- Tái sử dụng thành phần.
- Tách nghiệp vụ khỏi giao diện.
- Mọi form dùng React Hook Form và Zod.
- Mọi bảng dữ liệu phức tạp dùng TanStack Table.
