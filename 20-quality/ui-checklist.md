# UI Checklist // Danh sách kiểm tra giao diện

- [ ] Đúng màu token.
- [ ] Đúng thang khoảng cách.
- [ ] Đúng kích thước chữ.
- [ ] Không có inline style.
- [ ] Có loading, empty, error.
- [ ] Có trạng thái focus.
- [ ] PC, Tablet, Mobile hoạt động đúng.
- [ ] Mobile có vùng chạm tối thiểu 44x44px.
- [ ] Trạng thái có màu và chữ.
- [ ] Không lặp thành phần.
