# HPCons Design System // Hệ thống quy chuẩn thiết kế HPCons V1.0

Bộ tài liệu này là nguồn quy chuẩn chính thức cho toàn bộ hệ sinh thái phần mềm nội bộ HPCons.

## Phạm vi
- PC // máy tính
- Tablet // máy tính bảng
- Mobile // điện thoại
- Dark Mode // giao diện tối mặc định
- Enterprise UI // giao diện doanh nghiệp
- Data-dense // hiển thị dữ liệu mật độ cao

## Công nghệ ưu tiên
- Next.js // khung phát triển ứng dụng React
- React // thư viện xây dựng giao diện
- TypeScript // JavaScript có kiểm tra kiểu dữ liệu
- Tailwind CSS // bộ lớp định dạng giao diện
- shadcn/ui // thư viện thành phần giao diện
- TanStack Table // thư viện bảng dữ liệu
- React Hook Form // thư viện quản lý biểu mẫu
- Zod // thư viện kiểm tra dữ liệu
- Lucide Icons // bộ biểu tượng
- Framer Motion // thư viện chuyển động

## Cách dùng với Claude Code
1. Đặt toàn bộ thư mục này vào thư mục gốc của dự án.
2. Đảm bảo file `CLAUDE.md` nằm ở thư mục gốc dự án.
3. Khi giao việc, yêu cầu Claude Code đọc `CLAUDE.md` và tài liệu liên quan.
4. Không đưa toàn bộ tài liệu vào một câu lệnh duy nhất.
5. Mỗi nhiệm vụ chỉ nạp phần nền tảng, thành phần và mẫu nghiệp vụ cần thiết.

## Thứ tự đọc
1. `CLAUDE.md`
2. `01-project/README.md`
3. `02-foundations/README.md`
4. Tài liệu bố cục tương ứng thiết bị
5. Tài liệu thành phần liên quan
6. Tài liệu mẫu nghiệp vụ liên quan
7. Tài liệu quy tắc lập trình
